# AIZSH Game Demo

This repository contains a small side-scrolling shooter written in Python with `pygame`. The game is inspired by the classic shooter **Raiden 3**, but uses a horizontal orientation.

## Controls
- **Arrow Up/Down**: Move the player ship vertically
- **Space**: Shoot bullets

Enemies spawn from the right and move left across the screen. If an enemy collides with the player, the game ends. Destroy enemies to increase your score.

## Requirements
- Python 3
- `pygame` (`pip install pygame`)

## Running the Game
```
python3 main.py
```
